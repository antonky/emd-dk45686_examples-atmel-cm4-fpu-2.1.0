tmst_wom_config_wom_int_mode_t  wom_int,
tmst_wom_config_wom_int_dur_t  wom_dur 
)

Enable Wake On Motion.

Parameters
[in]sPointer to device.
[in]wom_x_thThreshold for X axis with 1g/256 resolution (wom_x_th = mg * 256 / 1000).
[in]wom_y_thThreshold for Y axis with 1g/256 resolution (wom_y_th = mg * 256 / 1000).
[in]wom_z_thThreshold for Z axis with 1g/256 resolution (wom_z_th = mg * 256 / 1000).
[in]wom_intMode used to generate interrupt (AND/OR).
[in]wom_durNumber of overthreshold events to wait before generating interrupt.
Returns
0 on success, negative value on error.

◆ inv_imu_adv_enable_wom()

int inv_imu_adv_enable_wom ( inv_imu_device_t *  s)

Enable Wake On Motion.

note : WoM requests to have the accelerometer enabled to work. As a consequence Fifo water-mark interrupt is disabled to only trigger WoM interrupts. To have good performance, it's recommended to set accelerometer ODR (Output Data Rate) to 20ms and the accelerometer in Low Power Mode.

Parameters
[in]sPointer to device.
Returns
0 on success, negative value on error.
